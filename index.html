What is crocheting

Crocheting is a craft that involves creating fabric by interlocking loops of yarn, thread, or other materials using a hooked needle, called a crochet hook. It is different from knitting in that each stitch is completed before starting the next one, and it typically uses only one hook, while knitting uses two needles. Crocheting can be used to make various items such as blankets, scarves, hats, and decorative items.

Stitches

Crocheting involves many types of stitches, each used to create different textures, patterns, and designs. Below is a list of some common crochet stitches, along with their abbreviations and descriptions. There are also variations and combinations that create even more unique stitches.

Basic crochet stitches

Chain Stitch (ch):
  • Description: The foundation for most crochet projects.
  • It creates a row of loops that you work into to start your first row.
  • How to: Yarn over (yo) and pull through the loop on the hook.
Slip Stitch (sl st):
  • Description: Used to join pieces or move the yarn without adding height to the fabric.
  • How to: Insert the hook into a stitch, yarn over, and pull through both the stitch and the loop on the hook.
Single Crochet (sc):
  • Description: A basic stitch that creates a dense and sturdy fabric.
  • How to: Insert the hook into a stitch, yarn over, pull up a loop, yarn over again, and pull through both loops on the hook.
Half Double Crochet (hdc):
  • Description: Taller than a single crochet but shorter than a double crochet.
  • How to: Yarn over, insert the hook into a stitch, yarn over, pull up a loop, yarn over again, and pull through all three loops on the hook.
Double Crochet (dc):
  • Description: Creates a taller, more open stitch.
  • How to: Yarn over, insert the hook into a stitch, yarn over, pull up a loop, yarn over again, pull through two loops, yarn over, and pull through the remaining two loops.
